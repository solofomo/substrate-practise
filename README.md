# substrate-practise

使用Rust语言写一个冒泡排序的算法
- ✅ 基础要求：固定类型（比如i32）的数组排序
- ✅ 提高部分：能够使用template和PartialOrd实现对任意类型的排序
- ✅ 使用cargo run执行
- ✅ 使用cargo test测试
